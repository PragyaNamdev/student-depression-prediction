# AI-Based Student Depression Prediction

This project predicts student depression using machine learning techniques based on various lifestyle and academic factors.

## 📊 Dataset
The dataset includes information about students such as:

- Gender
- Age
- Academic Pressure
- Work Pressure
- CGPA
- Study Satisfaction
- Sleep Duration
- Dietary Habits
- Work/Study Hours
- Financial Stress
- Family History of Mental Illness

## 🤖 Machine Learning Models Used

- Logistic Regression
- Random Forest

## 📈 Model Performance

| Model | Accuracy |
|------|---------|
| Logistic Regression | 83.69% |
| Random Forest | 82.79% |

## 📉 Evaluation Techniques

- Confusion Matrix
- Feature Importance Analysis
- Accuracy Comparison
